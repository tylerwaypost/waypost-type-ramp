# Waypost Typography System

Interactive type ramp and usage guidelines for Museo Sans Rounded.

**Deploy:** GitHub Pages (Settings → Pages → main branch) or drag folder to [app.netlify.com/drop](https://app.netlify.com/drop).

## What's included

- Type specimen with weight waterfall
- Complete digital scale reference (12 levels, px/rem/weight/leading/tracking)
- Weight system guide (300/500/700/900) with use cases
- Pairing & usage patterns (hero, content block, card)
- Responsive breakpoint table with clamp() implementations
- Print guidelines (sizing, leading, paper/ink considerations)

## Tech

Single `index.html` — no build step, no dependencies beyond Google Fonts. Uses Nunito as browser stand-in for Museo Sans Rounded (licensed via Adobe Fonts).
